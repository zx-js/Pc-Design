# 安装

> 我们不是 vue 的开发者，我们只是 vue 的搬运工

### npm 安装

```bash
npm i -D zx-util-ui
```

### yarn 安装

```bash
yarn add -D zx-util-ui
```

### vue 项目引入

> `import zui from 'zx-util-ui'`
