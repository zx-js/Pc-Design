# 安装

> 我们不是 vue 的开发者，我们只是 vue 的搬运工

::: explain npm 安装

```vue explain
npm i -S zx-pc-design
```

:::

---

::: explain yarn 安装

```vue explain
yarn add -S zx-pc-design
```

:::

---

::: explain vue 项目引入

```js explain
import zui from "zx-pc-design";
import "zx-pc-design/dist/z-style.css";

Vue.use(zui)
```

:::
