# 快速上手

::: explain 第一步 安装组件库

```vue explain
yarn add -S zx-pc-design
```

:::

---

::: explain 导入所有组件

```js explain
import zui from "zx-pc-design";
import "zx-pc-design/dist/z-style.css";

Vue.use(zui)
```

:::
